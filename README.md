# Website

This website is built using [Docusaurus 2](https://docusaurus.io/), a modern static website generator.

## 安装

```
$ npm install
```

### 测试

在本文档同级目录下运行：

```
$ npm run start 
```

会启动一个模拟服务器，并且在浏览器中打开预览页面，方便时间查看修改后的效果。


### 打包

```
$ npm run build
```

生成最终的静态网站内容。

### Deployment


修改完文档后直接提交即可。 **不需要** `npm run start` 或 `npm run build`。
